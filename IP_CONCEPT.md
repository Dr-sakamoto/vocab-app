# Etymon ― 世界観バイブル（独自IP）

> ブランド名 **Etymon（エティモン）** 本採用。御三家 **SPEC・PORT・VOC**（仮）。
> 最終決定で `lib/branding.js` を更新すれば全体へ反映する。

## 0. ブランド名「Etymon」について

**Etymon は実在の言語学用語**（複数 etyma/etymons）。「ある単語が派生してくる
**元の語・根の形＝語源そのもの**」を指す。ギリシャ語 étymon（語の*真の*姿）←
étymos（真実の・本物の）が語源で、**etymology（語源学）と同根**。
つまり「単語の本当の正体としてのモンスター」というコンセプトに完全一致し、
語尾に **-mon** を含むため**老若男女に「集める生き物」と一瞬で伝わる**。

## 1. ひとことで

**単語が、まるい生き物になる。** 英単語を思い出すと、その言葉が結晶化して
**Etymon** という生き物になる学習RPG。覚えるほど世界が開け、仲間が増える。

## 2. デザイン指針（最重要）

各参照の**特徴的な記号（＝パクればバレるもの）は使わない**。
その奥にある**原理だけ**を抽出してブレンドする。

### 2.2 オリジナリティの鉄則（パクリ元を消す）

> **テスト：完成キャラを見て「これ◯◯のパクリだ」と元ネタを名指しできたら失敗。**
> その場合、その元ネタから造形を*さらに引き離す*。

- 参照はブレンダーに入れる素材。出力は**どの単一ソースにも辿れない新種**であること。
- これは美学であると同時に**法的安全弁**：ポケモン侵害を抜けて、藤子／村上の侵害に
  落ちては本末転倒。固有モチーフ・特徴的造形の流用は禁止。
- 合成の核 = **まるくシンプルな体（普遍）＋スーパーフラットな平面色（確信）＋
  Y2Kの半透明ツヤ＋夢のように淡い発光**。この組み合わせ自体が Etymon の固有性。

### 2.3 大人にも効く射程（藤子F不二雄の根幹）

藤子F不二雄の本質は「子供向けの可愛さ」ではなく、**シンプルさの中に大人を唸らせる
深さ（SF異色短編のような“すこし・ふしぎ”）を同居させた射程の広さ**。
Etymon も、子供は可愛さで、大人はデザインの洗練と世界観の奥行きで愛せること。
→ シンプルな造形は、可愛くも・スタイリッシュにも・ほの哀しくも振れる**伸縮性**を持たせる。

> 合言葉：**王道なのに最先端。シンプルなのに誰も見たことがない。**
> まるく親しみやすい体＋スーパーフラットな色＋Y2Kのツヤ＋夢のように淡い発光。
> 背景は淡色のリミナルな夢空間。子供にも大人にも刺さり、元ネタは誰にも辿れない。

## 3. 賢い中身（メカニクスの骨格）

表面はやさしくポップだが、骨格は本物の言語構造：

- **単語＝生き物**：思い出した単語が結晶化して Etymon になる。
- **関連語＝進化（BLOOM）**：同じ語の仲間（派生語）を覚えると、その Etymon が
  次の姿へ**咲く（BLOOM）**。これは*実在の語源の枝分かれ*に対応（例: `PORT`→transport）。
  ただし世界の物語では「語源」と説明せず、「仲間の言葉を覚えると育つ」と自然に見せる。

→ 説得力（構造が本物）と、最先端ポップな見た目を両立させる。

## 4. 世界とプレイヤー

果てしなく淡い**夢のような空間**を、言葉が小さな光となって漂っている。
プレイヤーはその光を*思い出し*、形を与える者。難しい肩書きは付けず、
やさしいナビ役の**まるいマスコット**（ドラえもん的な、老若男女に好かれる相棒）が導く。
