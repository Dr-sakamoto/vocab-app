# 収益化ロードマップ

英単語学習RPG（ポケモン風）の収益化に向けた課題と実装計画。
本ドキュメントは全体の地図。IP差し替えの詳細は [`IP_REMEDIATION.md`](./IP_REMEDIATION.md) を参照。

## 採用する収益モデル（専門家としての決定）

**フリーミアム ＋ 月額サブスクを主軸**。理由：

- 学習アプリは**継続利用＝価値**であり、継続課金（サブスク）と相性が良い。
- 単発の買い切りは初速は出るが、継続収益・LTVが伸びにくい。
- ただしサブスク忌避層・贈与需要に応えるため、**ライフタイム買い切り（1回購入で恒久プレミアム）も併設**。
- **広告は無料層のみ**に表示し、プレミアムは広告非表示を内包（広告は補助的収益＋課金動機）。

### 無料 / プレミアムの線引き（実装済み: `lib/entitlements.js`）

| 機能 | 無料 | プレミアム |
| --- | --- | --- |
| 1日のプレイ回数 | 5セッション/日 | 無制限 |
| 単語プール上限 | 300語まで | 無制限 |
| 広告 | あり | なし |
| クラウド同期 | × | ○ |
| ストーリー全章 | 一部 | 全解放 |
| 上位捕獲アイテム等 | 制限 | 解放 |

> 数値は `lib/entitlements.js` の `LIMITS` / `PLAN_FEATURES` で一元管理。
> プロダクト判断で調整可能な「ダイヤル」。価格・閾値はリリース後の計測で最適化する。

価格は仮置き（例: 月額 ¥480 / ライフタイム ¥3,800）。**計測導入後にA/Bで決定**する。
