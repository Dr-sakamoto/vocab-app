# Vocab App — コンセプト & 開発指針

## アプリのコンセプト
ポケモンのバトル・捕獲システムをモチーフにした英単語学習アプリ。
「楽しく続けられること」が最優先。ゲーム体験を犠牲にした機能追加は行わない。

## コアバリュー（優先順位順）
1. **ゲームとしての面白さ** — バトル・捕獲・パーティ編成の楽しさを損なわない
2. **学習効果** — 単語が自然に身につくループ設計
3. **継続性** — 毎日開きたくなるモチベーション設計
4. **パフォーマンス** — 快適な操作感

## 技術スタック
- Next.js（App Router）
- Supabase（DB・認証）
- Tailwind CSS

## 開発の禁止事項
- ゲームのコアループ（バトル→結果→捕獲）を壊す変更
- 既存テスト（tests/）を壊す変更
- 環境変数・APIキーをクライアントサイドに露出させる実装
- 大規模リファクタリング（スコープを絞った小さな改善のみ許可）

## 自律AIエージェントへの追加指示
- 毎回必ずテストを実行して全パスを確認してからPRを作成する
- 1PRにつき1つの改善に絞る（複数混在させない）
- 迷ったら「何もしない」を選ぶ（壊すくらいなら手を付けない）
